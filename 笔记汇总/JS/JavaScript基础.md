# 4. 变量

### ① 声明变量+赋值

```
var age = 10;
```

### ② 处理输入

```
var myname = prompt("请输入你的姓名")
alert(myname);
```

### ③ 声明多个变量

```
var a = 1,
	b = 2,
	c = 3;
```

### ④ 特殊情况

```
1.只声明不赋值  为undefined
var sex；
console.log(sex)
2.不声明不赋值  报错

3.不声明、直接赋值 可以运行
```
